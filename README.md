*Gas pipeline route analysis for major energy infrastructure project*

**The Challenge:** Identify optimal onshore gas-line routes from offshore landfall to NLNG plant, analyzing environmental, engineering, and socio-cultural constraints.

**My Role:** 
- Analyzed 0.5m resolution satellite imagery for land-use/land-cover classification
- Processed vector datasets including settlements, powerlines, pipelines, and telecom infrastructure
- Generated statistical assessments for multiple route options (Options 4G, 4G2, 4G3 and 5E, 5E2)
- Evaluated criteria including community impact, vegetation traversed, and infrastructure crossings

**Key Results:**
